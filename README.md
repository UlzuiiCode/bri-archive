# BRI Archive Hub

Sistem manajemen arsip digital BRI untuk pengelolaan dokumen yang efisien dan aman.

## Fitur

- 📁 Manajemen arsip dokumen digital
- 👤 Manajemen pengguna dengan sistem approval
- 🔐 Autentikasi dan otorisasi berbasis role (Admin & Pegawai)
- 📊 Dashboard aktivitas dan statistik
- 🔍 Pencarian dan filter dokumen

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Storage)
